⭐ Muito Cobrado — Banco de Dados
Árvore B e B+
Regra de ouro: Quando a prova falar de índices em BD, a resposta é Árvore B ou B+ — 99% das vezes.
Característica
Detalhe
Balanceamento perfeito
Todas as folhas no mesmo nível — busca uniforme
Otimizada para disco
Nó = tamanho de uma "página" de disco (ex: 4KB)
Multivalorada
Cada nó tem várias chaves e vários filhos
Árvore B+ (diferencial): dados reais ficam APENAS nas folhas. Nós internos = só índices. Folhas são encadeadas → buscas sequenciais muito rápidas.

Exemplo: SELECT * FROM t WHERE id > 50 → percorre as folhas em sequência.
Heap Binária
Fila de Prioridade
Max-Heap e Min-Heap
Não é BST! Não existe regra de esquerda/direita.

Max-Heap: Pai sempre maior que os filhos → raiz é o maior elemento.
Min-Heap: Pai sempre menor que os filhos → raiz é o menor elemento.
"Qual estrutura permite pegar o maior/menor elemento rapidamente?" → Heap. Usado no Heapsort e em filas de prioridade (impressão, escalonamento de processos).

Códigos de Ordenamento - Java
Leia o Código
// bubble Sort — método bolha--
for (int i = 0; i < n-1; i++) {
  for (int j = 0; j < n-i-1; j++) {
    if (v[j] > v[j+1]) { // ← compara vizinhos
      int tmp = v[j];
      v[j] = v[j+1];
      v[j+1] = tmp;    // ← troca
    }
  }
}
Sinal identificador: dois fors encadeados + troca de vizinhos v[j] e v[j+1].
// Insertion Sort — sinal: while com j--
for (int i = 1; i < n; i++) {
  int key = v[i];
  int j = i - 1;
  while (j >= 0 && v[j] > key) {
    v[j+1] = v[j]; // ← empurra para direita
    j--;           // ← recua índice
  }
  v[j+1] = key;
}
Código — Selection Sort Java
O Menor Vai pro Início
// Selection Sort — sinal: acha o menor e troca SÓ no final
for (int i = 0; i < n-1; i++) {
  int min_idx = i;
  for (int j = i + 1; j < n; j++) {
    if (v[j] < v[min_idx]) { // ← apenas busca, não troca aqui
      min_idx = j;
    }
  }
  int tmp = v[min_idx];    // ← troca apenas 1 vez por passagem
  v[min_idx] = v[i];
  v[i] = tmp;
}
Sinal identificador: busca pelo menor valor no loop interno (v[j] < v[min_idx]) e faz a troca de posições apenas uma vez, no final do loop externo.
